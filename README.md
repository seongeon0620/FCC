##  🙌🏻 프로젝트 정보
[비동기 통신 기반 웹소켓과 뷰엔진을 활용한 식품 데이터 시각화]<br>
멋쟁이사자처럼 테킷 백엔드 스쿨 플러스 4조 팀프로젝트입니다.<br>
TastyTales는 냉장고 속 재료들을 관리하고, 재료들의 영양정보를 한눈에 확인할 수 있는 웹사이트입니다.
- 개발 기간 : 2023.11.02 ~ 2023.11.09 (7일)
